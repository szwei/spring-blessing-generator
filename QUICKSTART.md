# 快速开始指南

## 🚀 5分钟启动项目

### 步骤 1：安装依赖

```bash
cd spring-blessing-generator
npm install
```

### 步骤 2：启动开发服务器

```bash
npm run dev
```

### 步骤 3：访问应用

打开浏览器访问：http://localhost:3000

## 📝 使用说明

### 生成祝福语

1. **选择祝福类型**
   - 🎊 普通祝福
   - 👨‍👩‍👧‍👦 家人祝福
   - 💼 商务祝福
   - 💑 情人祝福
   - 👋 朋友祝福
   - 🌸 温馨祝福

2. **输入称呼（可选）**
   - 例如：张总、爸爸、亲爱的

3. **输入关键词（可选）**
   - 例如：健康、财富、幸福

4. **选择祝福风格**
   - 传统典雅
   - 现代时尚
   - 幽默风趣

5. **点击"生成祝福语"**

### 操作功能

- **复制**：一键复制到剪贴板
- **保存图片**：保存为精美图片（开发中）
- **分享**：分享到社交媒体
- **换一条**：随机生成新的祝福语

## 🎨 功能特点

✅ 多种祝福类型，满足不同场景
✅ 自定义称呼和关键词，个性化定制
✅ 三种祝福风格，传统与现代结合
✅ 一键复制，方便快捷
✅ 响应式设计，支持移动端
✅ 春节主题，视觉冲击力强
